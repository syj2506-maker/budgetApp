budgetApp

Python CSV 기반 가계부 CLI 앱 현황

지금까지 구현된 핵심 로직, 테스트 범위, 데이터 검증 결과를 한 페이지에 정리한 상태 리포트입니다. 현재는 CLI UI보다 핵심 데이터 처리 계층을 먼저 TDD로 완성해가는 단계입니다.

현재 브랜치 main, 최신 커밋 f41cdbf
핵심 검증 pytest tests/test_core.py -q 기준 14개 테스트 통과

구현된 기능

  • add_transaction: 거래를 복사해서 새 거래를 추가
  • get_balance: 수입과 지출의 amount 합산
  • filter_by_category: 카테고리 대소문자 무시 필터링
  • load_transactions_from_csv: utf-8-sig CSV 로드 및 amount 정수 변환
  • monthly_summary: 월별 수입, 지출, 순이익 집계

테스트 현황

  • step1_transactions.csv 기반 CSV 로드 테스트
  • step2_transactions.csv 기반 잔액 및 카테고리 필터 테스트
  • step3_transactions.csv 기반 월별 요약 테스트
  • 빈 리스트, 미존재 카테고리, 원본 리스트 독립성 검증

모든 핵심 기능은 테스트를 먼저 추가한 뒤 구현하는 TDD 흐름으로 진행했습니다.

월별 요약 예시

월 수입 지출 순이익
2025-01 405,037 -2,886,860 -2,481,823
2025-12 6,867,295 -3,540,137 3,327,158
2026-03 489,857 -3,301,374 -2,811,517

이 값들은 data/step3_transactions.csv를 읽어서 계산한 검증 기준입니다.
